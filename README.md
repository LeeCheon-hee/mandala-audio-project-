# 曼陀罗音频沉浸空间

Audio-Reactive Mandala — 音乐可视化解压体验。

## 技术栈

- p5.js 1.9.0
- 原生 HTML / CSS / JavaScript

## 运行方式

1. 使用本地服务器打开 `index.html`，或直接双击打开
2. 推荐使用 **Chrome** 或 **Edge** 72+
3. 点击「开启曼陀罗音频沉浸空间」→ 授权屏幕共享 + 系统音频 → 自动全屏

## 核心功能

- 四层曼陀罗粒子随音乐 bass/mid/treble 实时变化
- 鼓点触发圆环波纹与粒子斥力
- 中英双语、玻璃拟态 UI
- 沉浸式模式按钮（曼陀罗内可切换全屏）
